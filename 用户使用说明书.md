## 二、功能详解

### 2.1 首页导航

#### 位置

### 2.2 诗生画（输入古诗，生成绘画）

#### 功能介绍
输入古诗词或任何文字描述，AI会根据内容创作对应的中国风绘画作品。

#### 界面布局

```
┌─────────────────┬─────────────────┐
│  左侧：输入区域  │  右侧：画作展示  │
│                 │                 │
│  诗词输入框      │  [生成的画作]   │
│  (大文本框)      │                 │
│                 │   [收藏] [保存]  │
│  风格选择        │   [重新生成]    │
│  [6种风格]      │                 │
│                 │                 │
│  [生成画作]按钮  │                 │
└─────────────────┴─────────────────┘
```

#### 详细步骤

**步骤1：输入诗词**

方式一：手动输入
1. 找到左侧的大文本框（灰色边框）
2. 点击文本框内部
3. 输入您喜欢的古诗词或文字描述

示例文本：
```
床前明月光，疑是地上霜。
举头望明月，低头思故乡。
```

方式二：使用推荐诗词（推荐）
1. 右侧会出现"美育诗词推荐"浮动窗口
2. 点击窗口中显示的诗词内容
3. 诗词自动填入输入框，无需手打

**步骤2：选择绘画风格**

文本框下方有两行风格标签：

**爷爷/奶奶喜欢：**
- **水墨写意**：传统中国水墨画，黑白灰色调，写意风格
- **工笔细描**：精细的工笔画，色彩典雅，细节丰富
- **山水古风**：传统山水画，注重意境和留白

**我喜欢：**
- **Q版卡通**：可爱的卡通风格，色彩鲜艳
- **奇幻动漫**：梦幻的动漫风格，充满想象力
- **蜡笔手绘**：儿童蜡笔画风格，天真烂漫

**如何选择：**
1. 点击任意一个风格标签
2. 被选中的标签会变成橙色背景 + 白色文字
3. 只能选一个，点击其他会自动切换

**步骤3：生成画作**

1. 确认诗词和风格都已选好
2. 点击左下方的"生成画作"按钮（橙色大按钮）
3. 按钮文字变成"正在创作..."，同时出现旋转动画
4. 等待5-10秒（AI需要时间思考和绘画）
5. 右侧画框会出现生成的画作

**可能的等待时间：**
- 快速：5秒左右
- 正常：8秒左右
- 慢速：10-15秒（网络慢时）

**步骤4：操作生成的画作**

画作生成后，右上角出现三个按钮：

1. **收藏按钮**（爱心图标）
   - 点击后，作品保存到"我的收藏"
   - 会弹出提示"收藏成功！"

2. **保存按钮**（下载图标）
   - 点击后，画作在新窗口打开
   - 右键点击图片 → 选择"图片另存为"
   - 选择保存位置，输入文件名，点击保存

3. **重新生成按钮**（刷新图标）
   - 点击后，用相同诗词和风格重新生成
   - 每次生成的画作会略有不同（AI的随机性）

#### 特别说明

**诗词推荐窗口的使用**
- 窗口可以拖动：鼠标按住窗口任意位置，拖到喜欢的位置
- 自动切换：每5秒自动换一首诗
- 手动换诗：点击"换一首"按钮
- 暂停/播放：点击右上角的"⏸ 暂停"或"▶ 播放"按钮

**诗词窗口显示内容（大号时）：**
```

│                         │

│                         │
```

### 2.3 画生诗（上传图片，生成诗词）

#### 功能介绍
上传一张图片，AI会观察画面内容，创作对应的古风诗词。

#### 界面布局

```
┌─────────────────┬─────────────────┐
│  左侧：图片区域  │  右侧：诗词展示  │
│                 │                 │
│  [示例轮播]      │  [生成的诗词]   │
│  或              │                 │
│  [上传的图片]    │   [收藏] [复制]  │
│                 │   [重新生成]    │
│  诗词风格选择    │                 │
│  [4种风格]      │                 │
│                 │                 │
│  [生成诗词]按钮  │                 │
└─────────────────┴─────────────────┘
```

#### 详细步骤

**步骤1：准备图片**

方式一：使用示例图片（推荐新手）
1. 打开画生诗页面，左侧自动显示示例轮播
2. 看到5张精美的示例图片自动切换
3. 点击任意示例图片
4. 图片自动加载，可直接使用

**示例图片介绍：**
1. **山水意境**：云雾缭绕的山水水墨画
2. **春日花鸟**：桃花与喜鹊的工笔画
3. **秋月思乡**：月下庭院的意境图
4. **渔舟唱晚**：夕阳下的江面渔船
5. **雪梅傲骨**：雪中红梅的工笔画

**轮播控制：**
- 自动播放：每5秒切换一张
- 暂停/播放：点击右上角的"⏸ 暂停"或"▶ 播放"
- 手动切换：点击左右箭头按钮
- 跳转：点击底部的小圆点指示器

方式二：上传自己的图片
1. 点击绿色的"上传画作"按钮
2. 弹出文件选择窗口
3. 找到您的图片文件（支持JPG、PNG格式）
4. 双击图片或点击"打开"
5. 图片显示在左侧画框中

**上传后：**
- 示例轮播自动隐藏
- 只显示您上传的图片
- 按钮变成"重新上传"

**步骤2：选择诗词风格**

图片下方有4种诗词风格可选：

1. **古风诗**：古典风格的五言七言诗
   - 示例："远上寒山石径斜，白云生处有人家"
   - 特点：韵律工整，意境悠远

2. **词牌**：传统词牌风格创作
   - 示例：《水调歌头》《满江红》等格式
   - 特点：长短句结合，情感丰富

3. **绝句**：简洁优美的绝句风格
   - 示例："两个黄鹂鸣翠柳，一行白鹭上青天"
   - 特点：简洁精炼，意象鲜明

4. **赋体**：铺陈描写的赋体风格
   - 示例：《滕王阁序》式的华丽描写
   - 特点：辞藻华丽，气势磅礴

**如何选择：**
1. 点击任意一个风格标签
2. 被选中的标签会变成绿色背景 + 白色文字
3. 只能选一个风格

**步骤3：生成诗词**

1. 确认图片和风格都已选好
2. 点击左下方的"生成诗词"按钮（绿色大按钮）
3. 按钮文字变成"正在创作中..."，出现旋转动画
4. 等待5-15秒（AI需要识别图片并创作）
5. 右侧会显示生成的诗词

**生成时间说明：**
- 示例图片：5-8秒（已缓存）
- 上传图片：10-15秒（需先上传再识别）

**步骤4：查看和操作诗词**

诗词生成后的展示格式：
```
┌─────────────────────────┐
│  生成的诗词   [古风诗]   │
│  [收藏] [复制] [重新生成]│
│                         │
│      《秋江晚渡》        │
│                         │
│  残阳铺水漾金鳞，        │
│  归帆点点入烟津。        │
│  渔唱一声烟际起，        │
│  山容千叠黛中昏。        │
│                         │
│  波摇碎影随舟远，        │
│  风送余温伴客身。        │
│  此境浑然忘物我，        │
│  唯留暮色与闲心。        │
│                         │
└─────────────────────────┘
```

**操作按钮：**

1. **收藏按钮**（爱心图标）
   - 点击保存诗词和图片到收藏夹
   - 提示"收藏成功！"

2. **复制按钮**（复制图标）
   - 点击复制诗词文本到剪贴板
   - 可以粘贴到微信、邮件等

3. **重新生成按钮**（刷新图标）
   - 用相同图片和风格重新创作
   - 每次生成的诗词会有所不同

#### 特别说明

**如何重新上传图片：**
1. 点击"重新上传"按钮
2. 选择新的图片文件
3. 旧图片被替换，诗词清空

**如何返回示例模式：**
- 暂不支持（页面刷新后自动恢复示例）

### 2.4 诗词推荐窗口

#### 功能介绍
浮动在诗生画页面的诗词推荐小窗口，随机展示经典诗词，方便您快速选择创作素材。

#### 位置
- 默认位置：屏幕左下角
- 可自由拖动到任意位置

#### 如何拖动
1. 鼠标移到窗口任意位置
2. 按住鼠标左键不放
3. 拖动到想要的位置
4. 松开鼠标，窗口固定在新位置

**提示**：拖动时光标会变成"抓手"图标

#### 窗口内容（大号窗口）


#### 功能操作

**1. 自动切换**
- 每5秒自动换一首诗
- 进度条实时显示倒计时
- 切换时有淡入淡出动画

**2. 手动换诗**
- 点击"换一首"按钮
- 立即切换到新诗词
- 自动切换计时器重置

**3. 点击诗词使用**
- 点击诗词内容区域
- 诗词自动填入诗生画的输入框
- 方便快速创作

**4. 收藏诗词**
- 点击"收藏"按钮
- 弹出提示"已收藏《诗词名》"
- 保存到我的收藏

**5. 智能推荐**
- 最近显示过的5首诗不会重复
- 避免短时间内看到重复内容
- 推荐内容随机但不重复

#### 窗口大小调整

在系统设置中可以调整窗口大小（详见"系统设置"章节）：

**大号窗口**（默认）：
- 显示：诗词全文 + 配图 + 作者 + 赏析
- 适合：学习模式，慢慢品读

**中号窗口**：
- 显示：诗词全文 + 配图 + 作者
- 隐藏：赏析文字、操作按钮
- 适合：日常使用

**小号窗口**：
- 显示：仅诗词标题
- 隐藏：配图、全文、赏析
- 适合：节省空间，不遮挡操作

### 2.5 我的收藏

#### 功能介绍
查看您收藏的所有诗画作品，可以重新查看、下载或删除。

#### 界面布局

```
┌─────────────────────────────────┐
│          我的收藏                │
│   您收藏的精彩作品都在这里       │
├─────────────────────────────────┤
│                                 │
│  [作品1]  [作品2]  [作品3]      │
│                                 │
│  ┌──────────┐ ┌──────────┐     │
│  │  [图片]  │ │  [图片]  │     │
│  │          │ │          │     │
│  │  诗词...  │ │  诗词...  │     │
│  │          │ │          │     │
│  │ [删除]   │ │ [删除]   │     │
│  └──────────┘ └──────────┘     │
│                                 │
│  ... 更多作品 ...               │
│                                 │
└─────────────────────────────────┘
```

#### 如何查看收藏

**步骤1：进入收藏页面**
1. 点击顶部导航栏的"我的收藏"
2. 页面切换到收藏列表

**步骤2：浏览收藏**
1. 收藏作品按时间倒序排列（最新在前）
2. 每个作品显示：
   - 类型标签（诗生画/画生诗）
   - 图片或诗词内容
   - 风格标签
   - 收藏时间

**步骤3：操作作品**

每个收藏卡片上有操作按钮：

1. **查看详情**
   - 点击卡片主体区域
   - 弹出大图或完整诗词

2. **删除收藏**
   - 点击"删除"按钮
   - 确认后从收藏中移除
   - 无法恢复，请谨慎操作

3. **重新创作**（如果是诗生画）
   - 点击"再次生成"
   - 跳转到诗生画页面
   - 自动填入原诗词

#### 收藏数据说明

**存储位置**：
- 保存在您的浏览器本地
- 不会上传到服务器
- 隐私安全

**容量限制**：
- 约可存储50-100个作品
- 超出后提示清理旧收藏

**数据同步**：
- 不同浏览器的收藏不互通
- 清除浏览器数据会丢失收藏
- 建议定期导出重要作品

## 三、常见问题解答

### Q1：为什么点击"生成画作"没有反应？

**可能原因和解决方法：**

1. **未输入诗词**
   - 检查左侧文本框是否为空
   - 输入至少一句诗词

2. **网络问题**
   - 检查网络连接是否正常
   - 刷新页面重试

3. **AI服务繁忙**
   - 等待10-20秒再尝试
   - 高峰期（晚上8-10点）可能较慢

4. **浏览器兼容性**
   - 推荐使用Chrome、Safari、Edge
   - 更新浏览器到最新版本

---

### Q2：生成的画作可以保存到手机相册吗？

**是的，可以！**

**电脑端：**
1. 点击"保存"按钮
2. 图片在新窗口打开
3. 右键点击图片 → "图片另存为"
4. 选择保存位置

**手机端：**
1. 长按生成的图片
2. 选择"保存图片"
3. 图片自动保存到相册

---

### Q3：收藏的作品会永久保存吗？

**不一定，请注意：**

**数据存储位置**：
- 保存在浏览器本地（localStorage）
- 不会上传到云端

**可能丢失的情况**：
1. 清除浏览器缓存/数据
2. 卸载浏览器
3. 更换设备

**建议**：
- 重要作品及时下载保存
- 定期导出收藏（未来功能）
- 不要依赖浏览器长期存储

---

### Q4：诗词推荐窗口能关闭吗？

**目前不支持关闭，但可以：**

1. **拖动到屏幕边缘**
   - 拖到屏幕右下角
   - 拖到屏幕底部
   - 不遮挡主要操作区

2. **调整为小号窗口**
   - 进入系统设置
   - 选择"小"号窗口
   - 仅显示诗词标题，占用空间小

3. **切换到其他页面**
   - 诗词窗口仅在"诗生画"页面显示
   - 切换到"画生诗"或"收藏"页面
   - 窗口自动隐藏

---

### Q5：为什么上传图片后没有显示？

**排查步骤：**

1. **检查文件格式**
   - 仅支持JPG、PNG格式
   - 不支持GIF、BMP、WEBP

2. **检查文件大小**
   - 单个文件不超过5MB
   - 超出会提示"图片文件过大"

3. **检查图片完整性**
   - 确保图片未损坏
   - 尝试在其他应用打开

4. **浏览器问题**
   - 刷新页面重试
   - 清除浏览器缓存
   - 更换浏览器

---

### Q6：生成的诗词能修改吗?

**目前不支持直接修改，但可以：**

1. **重新生成**
   - 点击"重新生成"按钮
   - 每次生成内容会略有不同
   - 多生成几次选最好的

2. **复制后修改**
   - 点击"复制"按钮
   - 粘贴到文本编辑器（如记事本）
   - 自行修改润色

---

### Q7：可以批量生成吗？

**目前不支持批量生成。**

**每次只能生成一个作品：**
- 诗生画：一首诗 → 一幅画
- 画生诗：一张图 → 一首诗

**如需批量创作：**
1. 生成后立即收藏
2. 继续输入新内容
3. 重复操作

---

### Q8：为什么有时生成很慢？

**影响速度的因素：**

1. **网络速度**
   - WiFi信号弱
   - 网络带宽小
   - 解决：更换网络环境

2. **AI服务负载**
   - 高峰期使用人数多
   - 服务器处理慢
   - 解决：避开高峰期（晚8-10点）

3. **图片大小**（画生诗）
   - 上传大图片需要更长时间
   - 解决：压缩图片到1-2MB

**正常速度参考：**
- 诗生画：5-10秒
- 画生诗：10-15秒

---

### Q9：设置的背景图片模糊怎么办？

**原因：图片分辨率太低**

**解决方法：**
1. 使用高清图片
   - 推荐分辨率：1920x1080
   - 至少：1280x720

2. 重新上传
   - 删除当前背景（恢复默认）
   - 上传高分辨率图片

---

### Q10：忘记收藏在哪里了？

**查看收藏的步骤：**
1. 点击顶部导航栏的"我的收藏"
2. 进入收藏列表页面
3. 所有收藏作品按时间排列
4. 最新收藏在最上方

## 四、使用技巧

### 技巧1：如何获得更好的画作效果？

**输入技巧：**

1. **使用完整诗词**
   - 好：输入完整四句诗
   - 差：仅输入一两个词

2. **描述具体场景**
   - 好："月下荷塘，蛙声一片"
   - 差："好看的画"

3. **使用意象丰富的诗句**
   - 好："远山含黛，近水如蓝"（有颜色、景物）
   - 差："我很开心"（抽象情感）

4. **选择合适的风格**
   - 古诗词 → 水墨写意、工笔细描
   - 童话故事 → Q版卡通、奇幻动漫

---

### 技巧2：如何让AI生成更好的诗词？

**上传图片技巧：**

1. **选择主题明确的图片**
   - 好：山水画、花鸟图、人物肖像
   - 差：杂乱的街景、抽象艺术

2. **图片清晰度要高**
   - 好：高清照片、扫描件
   - 差：模糊、像素低的图片

3. **构图简洁**
   - 好：主体突出，背景干净
   - 差：元素太多，看不出重点

4. **选择合适的诗词风格**
   - 山水画 → 古风诗、绝句
   - 写意画 → 词牌
   - 工笔画 → 赋体

---

```
   ↓

```

## 五、注意事项

### 安全提示

1. **个人隐私**
   - 不要上传包含个人信息的图片
   - 不要上传他人隐私照片
   - 不要上传敏感内容

2. **版权意识**
   - AI生成的作品可自由使用
   - 不要上传有版权的图片用于商业
   - 尊重原创作者权益

3. **数据安全**
   - 定期备份重要收藏
   - 不要依赖浏览器长期存储
   - 清除浏览器数据前先导出

---

### 使用限制

1. **文件限制**
   - 图片大小：最大5MB
   - 图片格式：仅JPG、PNG
   - 背景图：最大5MB

2. **内容限制**
   - 诗词长度：建议100字以内
   - 不支持特殊符号（emoji等）

3. **功能限制**
   - 暂不支持视频生成
   - 暂不支持音频朗诵
   - 暂不支持3D效果
