# 诗画传情 - 使用说明书

**5分钟学会用，老人小孩都能上手**

---

## 打开软件

1. 浏览器输入网址（或点击桌面图标）
2. 看到首页，有4个按钮

```
┌────────────────────────────────────┐
│  [诗生画] [画生诗] [我的收藏]      │
└────────────────────────────────────┘
```

---

## 一、诗生画（输入诗词→生成画）

### 最简单的方法

1. **看右边飘着的诗词小窗口**
2. **点一首你喜欢的诗**（自动填进去了）
3. **选一个画风**（比如"水墨写意"）
4. **点"生成画"**
5. **等10秒**，画就出来了

### 示例

```
诗词小窗口显示：《静夜思》
                床前明月光，疑是地上霜...

点一下 → 自动填到左边输入框
选择"水墨写意" → 点"生成画"
等待... → 出来一幅月夜水墨画
```

### 如果想自己输入

1. 左边框框里直接打字
2. 输入任何古诗（或自己写的）
3. 后面步骤一样

### 画风怎么选？

**爷爷奶奶喜欢：**
- 水墨写意（淡雅）
- 工笔细描（精致）
- 山水古风（大气）

**我喜欢：**
- Q版卡通（可爱）
- 奇幻动漫（炫酷）
- 蜡笔手绘（童趣）

---

## 二、画生诗（上传图片→生成诗词）

### 看示例（不用上传）

1. 点"画生诗"
2. 自动播放示例图片（每5秒换一张）
3. 点图片，就能生成对应的诗词

### 上传自己的图片

1. **点绿色"上传画作"按钮**
2. **选一张照片**（风景、花鸟、人物都可以）
3. **选诗词类型**（古风诗、词牌、绝句、赋体）
4. **点"生成诗词"**
5. **等10秒**，诗就出来了

### 示例

```
上传一张荷花照片
→ 选"古风诗"
→ 点"生成诗词"
→ AI写出：
   "碧叶田田舒广袖，红葩点点映清流..."
```

---

## 三、诗词小窗口（右下角飘着的）

### 它能干什么？

- **自动推荐诗词**（不用自己想）
- **点一下就能用**（自动填进去）
- **可以拖动**（想放哪放哪）

### 怎么用？

1. **等它自己换**（每5秒一首）
2. **或点"换一首"按钮**
3. **看到喜欢的，点一下诗词内容**
4. **自动跳到诗生画页面，已经填好了**

### 怎么拖动？

鼠标按住窗口任意地方，拖到你喜欢的位置

### 嫌太大？

点右上角⚙️设置 → 选"小" → 保存

---

## 四、系统设置（右上角⚙️）

### 功能1：换背景

1. 点⚙️设置按钮
2. 点"上传新背景"
3. 选一张照片（自家院子、风景、全家福都行）
4. 点"保存设置"
5. 背景立刻换掉

**不喜欢了？** 点"恢复默认"

---

### 功能2：调窗口大小

**诗词小窗口太大挡事？**

1. 点⚙️设置
2. 选"小"（只显示标题）
3. 点"保存设置"

**想看完整诗词？**

选"大"（显示全部信息）

---

## 五、收藏

### 怎么收藏？

**诗生画：** 生成后，点"收藏"按钮
**画生诗：** 生成后，点"收藏"按钮
**诗词窗口：** 点"收藏"按钮

### 怎么查看？

点顶部"我的收藏"按钮

### 怎么删除？

收藏列表里，点"删除"

---

## 六、常见问题

### Q：诗词窗口挡住了怎么办？

**A：** 拖到角落，或者设置里选"小"

### Q：生成的画/诗不满意？

**A：** 点"重新生成"，AI会给你换一个

### Q：上传按钮点了没反应？

**A：** 再点一次，或者刷新页面

### Q：我的收藏会丢吗？

**A：** 不会，存在浏览器里。但是：
- 不要清除浏览器数据
- 换电脑会丢

### Q：能打印吗？

**A：** 右键图片 → 保存 → 去打印店

### Q：手机能用吗？

**A：** 能用，但电脑屏幕大，体验更好

## 七、快速上手：3个场景
